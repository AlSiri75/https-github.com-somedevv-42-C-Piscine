## Exercise 01 : ft_print_params

|               Exercise 01             |
|---------------------------------------|
|             ft_print_params                     |
| Turn-in directory : ex01/             |
| Files to turn in : ft_print_params.c            |
| Allowed functions : write              |

 ### Objective: 

• Here we are talking about a *program*, therefore it will have to have a function
main in your .c file.

• Write a program that displays the arguments passed on the command line.

• One per line, in the same order as on the command line.

• Must return all arguments except argv [0].

• Example:
<pre><code>$>./a.out test1 test2 test3
test1
test2
test3
$></pre></code>
